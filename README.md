## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Running the project

- npm install
- Add a .env file to the root dir
- Add the mongodb URI and PORT (optional) to an .env file
- Run `npm start` and `npm run start:dev`
- Import request collection into Postman
- Change {{baseUrl}} variable in the collection to point to your localhost and
  port
